# Node101

A collection of Node.js learning projects and examples covering various topics including:

- Basic Node.js scripts
- File system operations
- Express.js applications
- Error handling
- MVC architecture
- URL shortener
- Cookies and sessions
- EJS templating

## Projects

- **ErrorHandling/**: Error handling examples
- **Express101/**: Basic Express.js setup
- **Express102/**: Advanced Express.js with components
- **ExpressCookies/**: Cookie management
- **ExpressSession/**: Session management with EJS
- **MVC_Project/**: Model-View-Controller pattern
- **ShortURLGenerator/**: URL shortening service
- **Lecture\*/**: Various lecture examples

## Getting Started

Each project has its own package.json. Install dependencies with:

```bash
npm install
```

Run individual projects as specified in their respective directories
